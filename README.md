## Setup & Installation

1. **Clone the Repository:**
   ```
   git clone https://github.com/V-M-B/Portfolio.git
   cd Portfolio
   ```

2. **Install Dependencies:**
   ```
   npm install
   ```

3. **Start the Dev Server:**
   ```
   npm run dev
   ```
   Runs at `http://localhost:3000` by default.

4. **Build for Production:**
   ```
   npm run build
   npm start
   ```
